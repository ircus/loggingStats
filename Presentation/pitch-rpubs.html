Data analysis

  • Leading language in scientific community → large number of packages that supply algorithms coming from various scientific domains
  • Machine learning: stats, caret (outlier detection, model fitting, clustering)
  • Linguistics: lda (topic modelling), NLP, openNLP
  • Networks (graphs): sna (neighborhoods, shortest paths, etc.)

Visualization and analysis delivery

  • ggplot2 - one of the best static visualization packages in R
  • rCharts - adds interactivity to ggplots
  • venneuler, VennDiagram - venn diagrams
  • shiny - embed interactive components in reports to manipulate plotted data or implement shareable applications (such as Logging Statistics)
